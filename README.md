
# INSTAGRAM DOWNLOADER

This project demonstrates how to automate form submission, modal handling, and video download using PHP WebDriver.

PHP 7.0 or higher
Composer
Selenium Server
ChromeDriver or appropriate WebDriver for your browser
PHP cURL extension
## Requirements
- PHP 7.0 or higher
- Composer
- Selenium Server
- ChromeDriver or appropriate WebDriver for your browser
- PHP cURL extension
